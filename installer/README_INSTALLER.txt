WHAT IS OBSCURA?
----------------

Obscura is a local-first, privacy-by-architecture research browser
designed for private web research without tracking or data collection.

It is NOT a replacement for Chrome, Firefox, or Edge. It is a 
specialized tool for:

  • Private research without tracking
  • Reading web content without ads or JavaScript bloat  
  • Maximum privacy with no compromises

KEY FEATURES
------------

  ✓ 100% Local - No servers, no backend, no accounts
  ✓ Zero Data Collection - No history, no cookies, no logs
  ✓ Privacy by Architecture - Privacy rules enforced at core level
  ✓ Multiple Privacy Modes - Lite, Standard, and Tor modes
  ✓ Clean Reading Experience - No ads, no trackers

SYSTEM REQUIREMENTS
-------------------

  • Windows 10 or Windows 11 (64-bit)
  • 200 MB disk space
  • Optional: SearxNG for search (can use public instance)
  • Optional: Tor for anonymous browsing

AFTER INSTALLATION
------------------

1. Launch Obscura from the Start Menu or Desktop shortcut
2. Type a search query or URL in the address bar
3. Adjust privacy settings using the Settings button
4. Close the app when done - all session data is cleared

For help and documentation, visit:
https://github.com/Dasoam/Obscura
