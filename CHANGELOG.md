## [1.0.0] - 2024-12-24

### Added
- Private search via DuckDuckGo (default) or SearxNG (optional)
- Multi-tab browsing with independent navigation
- Three privacy modes: Lite, Standard, Tor
- Text-only renderer (default) and WebEngine renderer
- Settings dialog with persistent preferences
- Modern dark theme UI
- Windows installer (Inno Setup)
- Standalone portable EXE

### Privacy Features
- Zero telemetry and analytics
- No browsing history saved
- No cookies persisted
- No user accounts or tracking
- Localhost-only communication (127.0.0.1)
- Privacy enforced at Core level
